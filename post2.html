COVID 19 Tracker

This was a project I worked on in August - September 2021. The app displays Covid related statistics worldwide and is deployed on Google FireBase platform.

I used Material UI library for designing frontend and chart.js library to help in pictorial representation of data (graphs) and leaflet to render interactive map.

Open disease data API disease.sh was used to retrieve covid data from online sources.

Technologies & Tools:
  • - HTML
  • - CSS
  • - ReactJS
  • - Material UI
  • - disease.sh API
